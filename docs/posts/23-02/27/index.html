Доделать расчет для Красноярска

Либерт попросил добавить в расчет места размещения имеющихся подразделений.

Сделал расчет профиля прибытия и его анализ

Сам отчет переформатирован к следующему виду:

  1. Анализ складывающейся обстановки с учетом имеющихся мест дислокации
  2. Анализ расчетного изменения с учетом наилучших мест дислокации
  3. Анализ расчетного изменения с учетом предложенных мест дислокации
  4. Сравнение профилей прибытия
