Опросник для Хлоповских

Обсудили как должен работать простой опросник по определению профессиональной ориентации для выставки в Красноярске. Попробую сделать в виде PowerPoint, как самой демократичной.

В итоге написал заготовку презентации, разобрался с кодом VBA, который позволяет реализовать нужный функционал.

Источник: https://dzen.ru/a/WqJGz3mdnR6LgbIj

' Счет букв
Public cnt_A, cnt_B, cnt_C As Integer
' Стартовая функция
Public Sub test_start()
    ' Обнуляем суммы
    cnt_A = 0
    cnt_B = 0
    cnt_C = 0
    
    ActivePresentation.SlideShowWindow.View.Next
End Sub
' Функции даных ответов (суммируют количество полученных ответов)
Public Sub is_A()
    cnt_A = cnt_A + 1
    ActivePresentation.SlideShowWindow.View.Next
End Sub
Public Sub is_B()
    cnt_B = cnt_B + 1
    ActivePresentation.SlideShowWindow.View.Next
End Sub
Public Sub is_C()
    cnt_C = cnt_C + 1
    ActivePresentation.SlideShowWindow.View.Next
End Sub
' Итоговая функция.
' Пока только выводит количество, потом из этого количества будет делаться вывод о 
Public Sub TotalCounts()
    MsgBox "Всего получено ответов:" & vbNewLine & _
           "A: " & cnt_A & vbNewLine & _
           "B: " & cnt_B & vbNewLine & _
           "C: " & cnt_C
End Sub

Расчет оптимального размещения ПСЧ в Красноярске

Обсудили расчет требуемого количества пожарных подразделений в г.Красноясрк. У ГУ запрошены данные по выездам. В следующем виде:

Обязательно:

Дата и время выезда, Адрес, тип (пожар, загорание, ложный, подгорание и т.д.), ранг пожара, площадь пожара, кол-во погибших и пострадавших, подразделение (первое выезжавшее), временные показатели (сообщение, выезд, прибытия, 1 ствол, локализация, ликвидация, возвращение), кол-во привлекавшейся техники.

Желательно:

Материальный ущерб, географические координаты (если есть), кол-во личного состава, причина пожара, кол-во стволов, кол-во привлекавшейся техники (по типам), ко-во израсходованных ОТВ
